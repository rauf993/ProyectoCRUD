# Welcome to My First Java Project

This project is a CRUD  application developed in Java using NetBeans. 

## Features

- **Add New Users**
- **Display All Users**
- **Update User Information**

## Getting Started

### Prerequisites

- Java Development Kit (JDK)
- NetBeans IDE
- Workbench(MYSQL)
